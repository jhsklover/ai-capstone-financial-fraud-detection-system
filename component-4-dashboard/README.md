# Dashboard
**Owner:** Jack Sklover
## Description 
Takes all of the data across the Airtable base following the Case Management workflow and displays metrics based on the data. This includes how many records belong to each status (e.g. case_created, analyzed), the risk level statistics (high, critical), how many total records there are, how many of the records produced cases, and average anomaly scores. The data is displayed in a clean, neutral aesthetic and is the UI of the system, which financial analysts can use to understand the metrics and trends.
